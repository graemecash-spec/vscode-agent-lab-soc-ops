# 🎉 Soc Ops — Social Bingo

> **Break the ice. Make connections. Win the room.**

A fast, fun bingo game designed for in-person mixers and team events. Each player gets a unique board filled with prompts like *"has a hidden talent"* or *"speaks more than 2 languages"* — mingle, find matches, and race to get **5 in a row!**

---

## ✨ Features

- 🎲 **Unique boards** — randomized for every player on every game
- 🎯 **25 prompts** — a diverse mix to spark real conversations
- 🆓 **Free space** — center square always marked to get you started
- 🏆 **Instant win detection** — rows, columns & diagonals all count
- 📱 **Mobile-friendly** — works great on any device, no app needed
- 💾 **Auto-save** — game state persists across page refreshes

---

## 🚀 Getting Started

**Prerequisites:** [Node.js 22+](https://nodejs.org/)

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) and share the link with your team!

---

## 🛠️ Customize It

Want to tailor the prompts for your event? Edit `src/data/questions.ts` — add, remove, or replace any of the 24 prompts to match your audience.

---

## 📦 Build & Deploy

```bash
npm run build
```

Pushes to `main` deploy automatically to **GitHub Pages**.
